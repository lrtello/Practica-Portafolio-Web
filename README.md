# Proyecto de desarrollo de un portafolio web personal utilizando HTML, CSS y JavaScript.
# ---------------------------------------------------------------------------------------
# Este proyecto tiene como objetivo crear un portafolio web atractivo y funcional que muestre mis habilidades, proyectos y experiencia profesional.
# El portafolio incluye secciones como "Acerca de mí", "Proyectos", "Habilidades" y "Contacto".
# Además, se implementaron características interactivas utilizando JavaScript para mejorar la experiencia del usuario.
# El diseño del portafolio es responsivo, asegurando que se vea bien en dispositivos de diferentes tamaños de pantalla.
# Tecnologías utilizadas:
# - HTML5 para la estructura del sitio web.
# - CSS3 para el diseño y la presentación visual.
# - JavaScript para la interactividad y funcionalidades dinámicas.
# El proyecto se desarrolló siguiendo las mejores prácticas de desarrollo web, incluyendo la accesibilidad y la optimización para motores de búsqueda (SEO).
